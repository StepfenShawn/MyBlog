# MyBlog
This is a simple website build in python and mysql.  
# Features
* Signin and Signup
* Post blog  
* Count the visitors
* Comment
* Share (todo) 
* OAuth (todo)
* Search (todo)
* Vedio (todo)
* Category by machine learning (todo)

# build
* mysql 8.0
* Python 3.9.4
   
```
python app.py
```  
Then the server will start at http://127.0.0.1:9000
